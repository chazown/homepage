# 🛠️ 제작이야기 (Making Story)

> **"The Best Free Tools for Creators"**
> 창작 활동에 필요한 최고의 무료 도구들을 한곳에 모아둔 큐레이션 페이지입니다.

## 🌟 프로젝트 개요
이 페이지는 '누구나 쉽게 홈페이지와 콘텐츠를 만들 수 있다'는 메시지를 전달하기 위해 기획되었습니다. 개발부터 디자인, 영상 편집까지 실제 제작 환경에서 가장 유용한 무료 서비스들을 선별하여 소개합니다.

## 🛠️ 사용된 기술
- **Bootstrap 5:** 빠르고 현대적인 레이아웃 구성을 위해 사용.
- **Custom CSS:** 카드 호버(Hover) 효과 및 이미지 최적화(`object-fit`).
- **Always Free Philosophy:** 유료 결제 없이도 높은 퀄리티의 결과물을 낼 수 있는 도구들 위주로 구성.

## 📋 소개된 도구들
- **Development:** Github (버전 관리 및 협업)
- **Infrastructure:** Oracle Cloud (웹 호스팅 및 서버)
- **Design:** Canva (비주얼 디자인)
- **Video:** CapCut & Vrew (AI 기반 숏폼 및 자막 편집)
- **Education:** 직업훈련포털 HRD (커리어 지원)
